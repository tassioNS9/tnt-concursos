## Tarefa

Crie uma página de profile do usuário, onde terá o input file para adicionar a imagem do usuário, além dos campos de Nome Completo, email, data de nascimento, CPF (campo bloqueado só de visualização), telefone, além dos campos de senha, para alterar a senha, sendo senha atual, nova senha e confirmar nova senha

## Requisitos Técnicos

- Crie a rota de API para buscar e atualizar os dados do profile e a rota para atualizar a senha, utilize o react hook form e o zod para validar os campos.
