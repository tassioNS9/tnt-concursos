## Tarefa

Estou criando um agente para auxiliar o usuário nos estudos, trazendo os detalhes referente a cada alternativa da questão.

## Requisitos Técnicos

O usuário clicar em um botão referente a cada questão, o ID é passado para o agente identificar a questão e retornar para o usuário **SOMENTE** os detalhes
de cada alternativa o porque está errada e a alternativa correta. Siga como base a rota de API.
